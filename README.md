# 🏢 Company Website Clone

A responsive company website clone built using **HTML5** and **CSS3**. This project recreates a modern business website with multiple pages and a clean user interface.

## 🚀 Features

- Responsive design
- Modern UI layout
- Multi-page website
- Service showcase
- Portfolio section
- Pricing plans
- Contact page
- Organized project structure
